## Este es un repositorio de prueba para la evaluación de unidad 4 de makerlab

**Evaluacion-Markdown-DanielaAldas**

Proceso de configuración

1. Crear un nuevo repositorio
2. Publicar el repositorio
3. Abrir el repositorio en la web
4. Cambiar la visualización del repositorio a público
5. Abrir el repositorio en Visual Estudio Code
6. Crear un nuevo archivo llamado TEST.md
7. Copiar y pegar las preguntas en el archivo TEST.md
8. Responder las preguntas
9. Crear un summary
10. Cargar el repositorio
